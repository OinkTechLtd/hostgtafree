# HostGta Free Extension

Автоматическое продление бесплатных веб-серверов для хостинга **HostGta.ru** и **HostGta.com**.

## Основные функции
- **Авто-продление**: Расширение проверяет ваши серверы каждые 6 часов. Если до окончания срока остается 3 дня или меньше, оно автоматически продлевает сервер на 30 дней.
- **Push-уведомления**: Вы получите уведомление в браузере, когда сервер будет успешно продлен.
- **Авто-логин**: Использует вашу текущую сессию в браузере. Достаточно один раз войти в личный кабинет.
- **Современный интерфейс**: Удобный Popup для мониторинга статуса всех ваших серверов.

## Установка
1. Скачайте архив `HostGtaFree.zip` из [релизов](https://github.com/YOUR_USER/HostGtaFree/releases).
2. Распакуйте архив.
3. Откройте браузер и перейдите в `chrome://extensions/`.
4. Включите **Режим разработчика** (Developer mode).
5. Нажмите **Загрузить распакованное расширение** (Load unpacked) и выберите папку с расширением.

## Как это работает
Расширение работает в фоновом режиме. Вам не нужно держать открытой вкладку хостинга. Как только срок сервера подходит к критической отметке, расширение само отправит запрос на продление.
